# Olist Business Intelligence Suite

Projeto completo de Business Intelligence utilizando PostgreSQL, SQL, ETL, Modelagem Dimensional e Power BI.

## Objetivos

Transformar dados brutos do dataset Olist em dashboards executivos capazes de suportar decisões de negócio.

## Tecnologias

- PostgreSQL
- DBeaver
- SQL
- Power BI
- Python
- Git
- GitHub
